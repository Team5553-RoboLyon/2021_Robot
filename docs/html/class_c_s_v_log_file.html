Detailed Description

A CSVLogFile writes values to a csv file

For the CSVLogFile to write log informations, you must call Log() periodically.

Instantiate a LogFile passing in its prefix and its column headings.

If you want the file to be saved in a existing directory, you can add its path before the file prefix. Exemple : to save the file in a usb stick on the roborio ("/media/sda1/") : LogFile("/media/sda1/log").
